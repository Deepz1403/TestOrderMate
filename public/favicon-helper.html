Option 1 - Quick Solution:

Use an online favicon generator like favicon.io

  2. Download the generated favicon package
  3. Replace the files in your public directory
Option 2 - Manual Creation:

Create a simple text-based favicon:

  1. Open any image editor (Photoshop, GIMP, Canva)
  2. Create a 32x32 pixel image
  3. Use blue background (#3b82f6) with white "OM" text
  4. Save as favicon.ico

Files you need to create/replace:

  • public/favicon.ico - Main favicon (16x16, 32x32)
  • public/apple-touch-icon.png - Apple devices (180x180)
  • public/icon-192.png - Android (192x192)
  • public/icon-512.png - Large icon (512x512)
